Для встановлення: запустити install.sh або
$ mkdir build
$ cd build
$ cmake .. -G "Unix Makefiles"
$ make

Результати в results.graph.png

Для запуску програми:

$ python3 main.py <path_to_bin> <number_of_threads> <number_of_repetitions>

Де number_of_threads - може бути число, або межі (1-10 прожене алгоритм за кількістю потоків від 1 до 10)

Для запуску бінарго файлу треба вказати шлях до конфігураційного

$ ./lab_integral <path_to_config>

На 5-ти потоках працює дуже швидко в нашому випадку, адже позибка, менша, ніж 1.026 досягається швидше
Проблема в тому, що з багатьма потоками по-різному ставляться точки для обрахунку
З одним обмеженням - з однією кількість потоків менше ітерацій, з іншим - на іншій
